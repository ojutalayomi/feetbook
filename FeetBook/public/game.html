Rules

  1. Objective:
    The goal of the game is to correctly guess a randomly generated number within a specified range.
  2. Game Setup:
    The game will randomly generate a number between a predefined minimum and maximum value.
  3. Number Range:
    The range of numbers for guessing is between 1 and 100.
  4. Keyboard Shortcut:
      Enter - Start
      C - Continue
      N / Right-Arrow - Next
      P / Left-Arrow - Previous Level / Rules
      S - Start
  5. Number of Attempts/Tries:
    Players are allowed a maximum of 10 attempts/tries to guess the correct number for Level 1. The number of attempts/tries depends on the level.
  6. Feedback for Guesses:
    After each guess, players will receive feedback indicating whether their guess is wrong or correct.
  7. Scoring:
    scoring system is implemented for this version of the game.
  8. Winning Conditions:
    Players win the game by correctly guessing the random number within the allotted number of attempts/tries.
  9. Difficulty Levels:
    The game has a 5 different level with specified number range and attempt limit.
  10. Time Limits:
    There is no time limit for making guesses.
  11. User Interface:
    Players interact with the game through a user-friendly interface on their chosen platform (web, mobile, etc.).
  12. Restarting the Game:
    Players have the option to restart the game after completion to try again.
  13. Penalties:
    If a player attempts to guess beyond the allotted number of attempts/tries, the game concludes, and the player is encouraged to restart.
  14. Fair Play:
    Fair play is encouraged, and any form of cheating or exploiting the game will result in disqualification.
  15. Accessibility:
    The game is designed to be accessible to a broad audience, including those with disabilities.
